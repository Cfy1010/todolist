# TodoList – exercice de révision JavaScript

## Contexte
Projet simple réalisé dans un cadre de révisions, visant à consolider les fondamentaux du développement web.

## Objectif
Mettre en œuvre une fonctionnalité basique de gestion de tâches (to-do list).

## Fonctionnalités
- Ajout, modification et suppression de tâches
- Interface volontairement simple pour tester les interactions utilisateur

## Technologies utilisées
- HTML
- CSS
- JavaScript

## Ce que j’ai travaillé
- Structuration d’une fonctionnalité fonctionnelle
- Lien entre interface utilisateur et logique métier
- Gestion des interactions utilisateur en JavaScript

## Pistes d’amélioration
- Persistance des données (localStorage ou backend)
- Amélioration de l’expérience utilisateur
- Mise en place de tests
